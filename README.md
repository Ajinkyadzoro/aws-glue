# AWS Glue Tutorial

* ApplyMapping
* DropFields
* DropNullFields
* Filter
* Join
* Map
* MapToCollection
* Relationalize
* RenameField
* ResolveChoice
* SelectFromCollection
* SelectFields
* Spigot
* SpliFields
* SplitRows
* Unbox
